Netdata is an open-source, real-time infrastructure monitoring platform. Monitor, detect, and act across your entire infrastructure.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
netdata_enabled: true
```

By default, Netdata can be accessed at `http://[your_server_ip]:19999` or `https://netdata.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Netdata at [`roles/netdata/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/netdata/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
