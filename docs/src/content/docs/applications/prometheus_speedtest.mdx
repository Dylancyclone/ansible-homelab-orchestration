Speedtest exporter is a [Prometheus](../prometheus) exporter written in Python using the official [Speedtest CLI](https://www.speedtest.net/apps/cli) made by Ookla.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
prometheus_speedtest_enabled: true
```

By default, Prometheus Speedtest can be accessed at `http://[your_server_ip]:9798` or `https://prometheus_speedtest.[your_domain].com` if you have [DNS access](../guides/dns-access) configured.

## Configuration

See the default configuration options for Prometheus Speedtest at [`roles/prometheus_speedtest/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/prometheus_speedtest/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
