 Media server for comics/mangas/BDs/magazines/eBooks with API, OPDS, Kobo Sync and KOReader Sync support 

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
komga_enabled: true
```

By default, Komga can be accessed at `http://[your_server_ip]:8088` or `https://komga.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Komga at [`roles/komga/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/komga/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
