FreshRSS is a self-hosted RSS and Atom feed aggregator. It is lightweight, easy to work with, powerful, and customizable.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
freshrss_enabled: true
```

By default, FreshRSS can be accessed at `http://[your_server_ip]:8089` or `https://freshrss.[your_domain].com` if you have [DNS access](../guides/dns-access) configured.

## Configuration

See the default configuration options for FreshRSS at [`roles/freshrss/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/freshrss/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
