Yamtrack is a self hosted media tracker for movies, tv shows, anime, manga, video games and books.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
yamtrack_enabled: true
```

By default, Yamtrack can be accessed at `http://[your_server_ip]:9008` or `https://yamtrack.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Yamtrack at [`roles/yamtrack/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/yamtrack/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
