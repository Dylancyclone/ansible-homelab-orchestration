Miniflux is a minimalist and opinionated feed reader. It's simple, fast, and lightweight.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
miniflux_enabled: true
```

By default, Miniflux can be accessed at `http://[your_server_ip]:8070` or `https://miniflux.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Miniflux at [`roles/miniflux/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/miniflux/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
