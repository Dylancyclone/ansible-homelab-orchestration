The open and composable observability and data visualization platform. Visualize metrics, logs, and traces from multiple sources like [Prometheus](../prometheus/), [Loki](../loki/), Elasticsearch, [InfluxDB](../telegraf/), Postgres and many more.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
grafana_enabled: true
```

The default login is `admin`/`admin` and you will be prompted to change the password on first login.

By default, Grafana can be accessed at `http://[your_server_ip]:3000` or `https://grafana.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Grafana at [`roles/grafana/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/grafana/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
