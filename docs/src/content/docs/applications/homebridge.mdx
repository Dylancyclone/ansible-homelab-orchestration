Homebridge allows you to integrate with smart home devices that do not natively support HomeKit. There are over 2,000 Homebridge plugins supporting thousands of different smart accessories.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
homebridge_enabled: true
```

By default, Homebridge can be accessed at `http://[your_server_ip]:8087` or `https://homebridge.[your_domain].com` if you have [DNS access](../guides/dns-access) configured.

## Configuration

See the default configuration options for Homebridge at [`roles/homebridge/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/homebridge/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
