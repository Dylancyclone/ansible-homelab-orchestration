Ubooquity is a free, lightweight and easy-to-use home server for your comics and ebooks. Use it to access your files from anywhere, with a tablet, an e-reader, a phone or a computer.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
ubooquity_enabled: true
```

To complete set up, access the admin page at `http://[your_server_ip]:2203/ubooquity/admin` and set a password.
Then you can access the web UI at `http://[your_server_ip]:2202/ubooquity/`

By default, Ubooquity can be accessed at `http://[your_server_ip]:2202` or `https://ubooquity.[your_domain].com` if you have [DNS access](../guides/dns-access) configured.

## Configuration

See the default configuration options for Ubooquity at [`roles/ubooquity/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/ubooquity/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
