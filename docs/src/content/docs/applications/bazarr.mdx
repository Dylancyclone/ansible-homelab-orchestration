Bazarr is a companion application to [Sonarr](../sonarr/) and [Radarr](../radarr/) that manages and downloads subtitles based on your requirements. 

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
bazarr_enabled: true
```

By default, Bazzar can be accessed at `http://[your_server_ip]:6767` or `https://bazarr.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Bazzar at [`roles/bazarr/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/bazarr/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
