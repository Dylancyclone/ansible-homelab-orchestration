Portainer is a lightweight management UI that allows you to easily manage your Docker environments.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
portainer_enabled: true
```

By default, Portainer can be accessed at `http://[your_server_ip]:9000` or `https://portainer.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for Portainer at [`roles/portainer/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/portainer/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
