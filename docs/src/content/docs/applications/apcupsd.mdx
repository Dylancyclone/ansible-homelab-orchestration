Dockerize `apcupsd`, but keep the ability to power off the docker host in event of a power failure.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
apcupsd_enabled: true
```

By default, Apcupsd can be accessed at `http://[your_server_ip]:3551` or `https://apcupsd.[your_domain].com` if you have [DNS access](../guides/dns-access) configured.

## Configuration

See the default configuration options for Apcupsd at [`roles/apcupsd/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/apcupsd/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
