The free and open-source Download Manager written in pure Python.

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
pyload_enabled: true
```

By default, pyLoad can be accessed at `http://[your_server_ip]:8000` or `https://pyload.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

## Configuration

See the default configuration options for pyLoad at [`roles/pyload/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/pyload/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
