Seerr is a free and open source software application for managing requests for your media library. It integrates with the media server of your choice: [Jellyfin](../jellyfin/), [Plex](../plex/), and [Emby](../emby/). In addition, it integrates with your existing services, such as [Sonarr](../sonarr/), [Radarr](../radarr/).

## Setting up

```yaml title="inventories/homelab/group_vars/homelab.yml"
seerr_enabled: true
```

By default, Seerr can be accessed at `http://[your_server_ip]:5055` or `https://seerr.[your_domain].com` if you have [DNS access](../../guides/dns-access/) configured.

### Migrating from Overseerr

Check out the [Overseerr page](../../archived_applications/overseerr/) for more information.

## Configuration

See the default configuration options for Seerr at [`roles/seerr/defaults/main.yml`](https://github.com/Dylancyclone/ansible-homelab-orchestration/blob/main/roles/seerr/defaults/main.yml).
Add any overrides to your `inventories/[your_inventory]/group_vars/homelab.yml` file.
